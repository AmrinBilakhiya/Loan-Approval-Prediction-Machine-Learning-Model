# Loan Approval Prediction Machine Learning Model 

This project is a **Loan Approval Prediction System** built using Python and Machine Learning. It helps banks and financial institutions automate the loan approval process by predicting whether a loan should be approved or not based on applicant data.

---

##  Project Overview

The goal of this project is to:
- Automate loan approval decisions
- Reduce manual effort
- Improve speed and accuracy in processing applications

The model is trained on historical loan data and predicts outcomes for new applicants.

---

##  Machine Learning Workflow

This project covers the complete ML lifecycle:

1. **Data Preprocessing**
   - Handling missing values
   - Encoding categorical variables
   - Feature selection

2. **Exploratory Data Analysis (EDA)**
   - Understanding patterns and distributions
   - Identifying important features

3. **Model Building**
   - Splitting dataset into training & testing sets
   - Training classification model

4. **Model Evaluation**
   - Testing accuracy and performance

5. **Prediction**
   - Predict loan approval for new inputs

6. **Model Saving**
   - Saving trained model using `pickle`

7. **Deployment**
   - Built a simple web app for real-time predictions
